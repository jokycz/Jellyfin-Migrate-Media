# Jellyfin Migrate Media

Work-in-progress tool for migrating a Jellyfin media library.

## What it does
- Moves **movies** and **TV series** from old library locations to new locations
- Rebuilds **folder hierarchy** and updates related **Jellyfin database records**
- Explores consistent remapping of items when reorganizing media storage

## Status
⚠️ **WIP / Proof of Concept**  
This repository is primarily a **code sample** and pre-screening preview.  
It is not feature-complete.

## Tech stack
- .NET (C#)
- SQLite (Jellyfin database)

## Notes
- Focused on understanding and experimenting with Jellyfin internal data structures
- Intended for learning, prototyping and demonstrating coding style
